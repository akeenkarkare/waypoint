# Waypoint

An AI-powered travel memory companion. Upload photos, add a caption or voice
note, jot a few thoughts — Waypoint weaves people, places, activities, emotions,
and experiences into a persistent **memory graph** so you can later ask
*"Tell me about my Japan trip"* or *"Where was that café I loved?"* and relive
your journeys as connected stories, not disconnected images.

Built on [Cognee](https://github.com/topoteretes/cognee) for the memory layer.

## Structure

```
backend/    FastAPI + Cognee memory graph  (see backend/README.md)
frontend/   Mobile app (React Native + Expo)
```
